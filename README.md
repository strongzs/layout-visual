> web布局可视化

## 安装

```sh
npm install @cjm/layout-visual --save
```

## 使用

```js
<template>
  <div>
    <vue-component />
  </div>
</template>

<script>
import VueComponent from '@cjm/layout-visual'

export default {
  components: {
    VueComponent
  }
}
</script>
```
